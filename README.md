# MBTR experiments
This repo contains the experiments performed in *Lorenzo Nespoli and Vasco Medici (2020).
Multivariate Boosted Trees and Applications to Forecasting and Control*
[arXiv](https://arxiv.org/abs/2003.03835)

    @article{nespoli2020multivariate,

    }

The datasets are directly downloaded from Zenodo. The code includes the figures presented in the paper, including the statistical tests.
